# Convex Backend

This directory contains the Convex backend for BLK Exchange.

## Structure

- `schema.ts` - Database schema (added in Task 2)
- `market.ts` - Market queries and mutations (added in Task 4)
- `players.ts` - Player mutations (added in Task 4)
- `_generated/` - Auto-generated by Convex CLI (do not edit)

## Setup

1. Install Convex CLI: `pnpm add -g convex`
2. Run `npx convex dev` after adding NEXT_PUBLIC_CONVEX_URL to .env.local
