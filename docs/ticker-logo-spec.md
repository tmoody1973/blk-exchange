# BLK Exchange Ticker Logo Spec

## For: Nanobanana AI Image Generation

Generate 36 company logos + 1 BLK Exchange master logo for use in-app, on OG share cards, and as merch-ready designs.

---

## Global Design Rules

- **Style**: Minimalist, neobrutalist. Flat design, no gradients, no 3D, no photorealism.
- **Shape**: Square format, 1024x1024. Logo mark centered with generous padding.
- **Background**: Solid black (#0A0A0A) for all logos.
- **Primary color**: White (#FFFFFF) for the logo mark.
- **Accent color**: One per sector (see below). Used sparingly as a single accent element.
- **Typography**: If the logo includes text, use a bold condensed monospace or geometric sans-serif. The ticker symbol (e.g., LOUD) should be secondary to the icon mark.
- **Vibe**: These should look like real brand logos, not clip art. Think Nike swoosh simplicity. Each should work at 32px (app favicon size) and on a hoodie at 8 inches.
- **No text in the logo mark itself** unless specified. The ticker symbol goes below or beside the mark, not inside it.
- **Every logo must be recognizable at thumbnail size** (64x64).

## Merch Application Guide

Each logo should work in these contexts:

| Application | Size | Notes |
|------------|------|-------|
| App icon (in ticker table) | 32x32 px | Must be recognizable |
| OG share card | 120x120 px | Alongside ticker name |
| Hoodie chest print | 8" wide | White on black, accent color optional |
| T-shirt back print | 12" wide | Logo + slogan + "blkexchange.com" |
| Cap front | 3" wide | Logo mark only, no text |
| Sticker | 3" circle | Logo mark + ticker symbol |
